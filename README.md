\# Context-Aware Music Recommender System



\## Overview

A machine learning-based music recommendation system that suggests songs based on user mood and audio features using unsupervised learning techniques.



\## Features

\- Mood-based music recommendations

\- KMeans clustering for song grouping

\- Feature-based similarity matching

\- Interactive Streamlit web app



\## Tech Stack

\- Python

\- Pandas

\- Scikit-learn

\- Streamlit



\## ML Approach

\- Feature scaling (StandardScaler)

\- Unsupervised learning (KMeans clustering)

\- Cosine similarity for ranking recommendations



\## How to Run

```bash

pip install -r requirements.txt

streamlit run app.py
